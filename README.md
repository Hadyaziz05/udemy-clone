# Udemy Clone

A React application built with Vite.

## Getting Started

### Prerequisites

- Node.js 20.19+ or 22.12+
- npm 8.0.0+

### Installation

Dependencies are already installed. To reinstall:

```bash
npm install
```

### Development

The development server is running at http://localhost:5173/

To start the dev server manually:

```bash
npm run dev
```

### Build

To create a production build:

```bash
npm run build
```

### Preview

To preview the production build:

```bash
npm run preview
```

### Lint

To run ESLint:

```bash
npm run lint
```

## Tech Stack

- React 19.2
- Vite 7.2
- ESLint 9.39
